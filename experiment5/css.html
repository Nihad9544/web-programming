What is CSS?

CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe how HTML elements should look on a web page.

three type css

  1. Inline
  2. Internal
  3. external

Why use CSS?

  • Add colors, fonts, and layouts to your website.
  • Make your site look attractive and consistent.
  • Control the design of multiple pages at once.
  • Make websites responsive (look good on phones, tablets, and computers).
